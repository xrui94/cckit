# UUID 测试说明

## 测试概述

本目录包含 UUID 子库的测试用例，涵盖基础功能测试和性能测试。

## 测试文件

### 1. test_uuid_basic.cpp
基础功能测试，验证 UUID 的核心功能：

- **nil UUID 测试**：验证默认构造和 nil UUID 的行为
- **UUID 生成测试**：验证 UUID 生成的唯一性
- **字符串转换测试**：验证 UUID 与字符串之间的转换
- **比较运算符测试**：验证相等、不等和小于运算符
- **哈希函数测试**：验证 std::hash 特化和容器支持
- **POD 转换测试**：验证与底层 C API 的互操作性
- **便捷函数测试**：验证便捷函数 genUuid()
- **拷贝和赋值测试**：验证对象的拷贝语义
- **边界情况测试**：验证无效输入的处理

### 2. test_uuid_performance.cpp
性能测试，评估 UUID 操作的性能：

- **生成性能测试**：评估 UUID 生成的速度
- **字符串转换性能测试**：评估 toString() 和 fromString() 的性能
- **比较性能测试**：评估比较运算符的性能
- **哈希性能测试**：评估哈希函数的计算速度
- **容器性能测试**：评估在 std::map、std::unordered_map 和 std::set 中的性能

## 编译和运行

### 编译测试

```bash
# 配置项目
cmake -B build -DCCKIT_BUILD_TESTS=ON

# 编译
cmake --build build
```

### 运行所有测试

```bash
# 进入构建目录
cd build

# 运行所有测试
ctest --output-on-failure
```

### 运行单个测试

```bash
# 运行基础功能测试
./test_uuid_basic

# 运行性能测试
./test_uuid_performance

# 或者使用 CTest
ctest -R uuid_basic
ctest -R uuid_performance
```

## 注意事项

1. **性能测试结果可能因硬件而异**
2. **测试运行时间可能较长，特别是性能测试**
3. **建议在 Release 模式下运行性能测试以获得准确结果**
4. **基础功能测试在 Debug 和 Release 模式下都应该通过**
